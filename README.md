# NBA-playerlist sovellus
Sovellukseen voi listata NBA koripallo pelaajia. Sovelluksessa onnistuu myös seuraavat asiat:

- Kirjautuminen: user ja admin roolit
- Uuden pelaajan lisääminen molemmilla rooleilla
- Muokkaaminen ja poistaminen admin roolilla
- Uloskirjautuminen

## Tietokanta
- H2
- MySql

## Pilvipalvelu
- Rahti
